index.php (raíz del sitio)
  └─ require wp-blog-header.php
       ├─ require_once wp-load.php
       │    ├─ define ABSPATH
       │    ├─ require_once wp-config.php
       │    │    └─ define constantes (DB, salts, WP_DEBUG...)
       │    └─ require_once wp-settings.php
       │         ├─ Conectar a la BD (wpdb)
       │         ├─ Cargar must-use plugins (wp-content/mu-plugins/)
       │         ├─ Cargar active plugins (wp_get_active_and_valid_plugins)
       │         │    └─ do_action('plugins_loaded')
       │         ├─ Cargar el tema activo (functions.php)
       │         │    └─ do_action('after_setup_theme')
       │         ├─ do_action('init')
       │         └─ do_action('wp_loaded')
       │
       └─ require_once wp-includes/template-loader.php
            ├─ do_action('wp') — query principal ejecutada
            ├─ do_action('template_redirect')
            └─ Cargar el template según Template Hierarchy
                 └─ do_action('wp_head') ... do_action('wp_footer')
