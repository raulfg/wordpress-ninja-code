# Nombre del Sitio

Brief descripción del proyecto y cliente.

## Stack

- WordPress 6.5 + Bedrock
- PHP 8.2 / MySQL 8.0 / Nginx
- Tema: NinjaTheme (en `/web/app/themes/ninjatheme/`)
- Plugins relevantes: ACF Pro, Polylang, WooCommerce
- Object cache: Redis
- CDN: Cloudflare
