# WordPress: de Cero a Ninja — Ejemplos de código

Repositorio de acompañamiento del libro **WordPress: de Cero a Ninja** (Raul Fernández).

Cada carpeta corresponde a un capítulo del libro. Los archivos contienen los ejemplos completos que el libro referencia con la nota:

```
// Código completo: github.com/raulfg/wordpress-ninja-code
```

## Uso

El código está pensado para ejecutarse tal como aparece, con las dependencias indicadas en cada capítulo del libro. Los requisitos mínimos son PHP 8.1 y WordPress 6.4 salvo que el archivo indique lo contrario.
